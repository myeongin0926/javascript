js 내장 객체, 메서드 문자열의 길이( length ) length 프로퍼티에는 문자열의 길이가 저장됩니다. length는 프로퍼티 입니다. str.length()가 아닌 str.length 함수가 아닌 숫자가 저장되는 프로퍼티이기에 괄호를 붙이지 않습니다 length를 배열에 사용할 경우 배열내 요소들의 개수를 나타냅니다. 특정 글자에 접근하기( charAt( ) , [pos] ) 문자열 내 특정 위치에 있는 글자에 접근하려면 [pos]같이 대괄호를 이용하거나 str.charAt(pos)라는 메서드를 호출하면 됩니다. 위치는 0부터 시작합니다. 근래에는 대괄호를 이용하는 방식을 사용합니다. charAt은 하위 호환성을 위해 남아있는 메서드라고 생각하시면 됩니다. 두 접근 방식의 차이는 반환할 글자가 없을 때 드러납니다. 접근하려는 위치에 글자가 없는 경우 []는 undefined를, charAt은 빈 문자열을 반환합니다. 소문자 변경( toLowerCase( ) ) toLowerCase() 메서드는 소문자로 변환된 호출 문자열 값을 반환합니다. 이 toLowerCase()메서드는 소문자로 변환된 문자열 값을 반환합니다. toLowerCase()는 문자열 자체 en의 값에는 영향을 주지 않습니다. 대문자 변경( toUpperCase( ) ) toUpperCase() 메서드는 대문자로 변환된 호출 문자열 값을 반환합니다. 이 toUpperCase()메서드는 대문자로 변환된 문자열 값을 반환합니다. toUpperCase()는 문자열 자체 en의 값에는 영향을 주지 않습니다. 문자열 여부 확인( includes() ) arr.includes(valueToFind[, fromIndex]) 매개변수 valueToFind : 탐색할 요소 - includes()는 대소문자를 구분합니다. fromIndex : 이 배열에서 검색을 시작할 위치입니다. - 기본값은 0입니다. 반환값 boolean - true,false 예제 부분 문자열 추출( slice() ) slice() 메서드는 어떤 배열의 begin 부터 end 까지(end 미포함)에 대한 얕은 복사본을 새로운 배열 객체로 반환합니다. 원본 배열은 바뀌지 않습니다. 매개변수 begin : 추출 시작점을 나타내는 부분입니다. 음수 인덱스는 배열의 끝에서부터의 길이를 나타냅니다. begin이 배열의 길이보다 큰 경우에는 빈 배열을 반환합니다. end : 추출 종료점을 의미합니다. slice()는 end인덱스를 제외하고 추출합니다. end가 생략되면 배열의 끝까지 추출합니다. 반환값 추출을 완료한뒤 그 요소를 담은 배열 설명 slice()는 원본을 대처하지 않습니다. 이 말은 즉슨 원본값을 변경하지 않고 옅은 복사본을 반환합니다.
